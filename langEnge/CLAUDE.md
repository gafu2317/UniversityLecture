### **課題ごとの詳細To-Doリスト**

#### **【課題1】**
*   **1ページ以上**書くこと。
*   自然言語処理の手法である「文法や事例等のルールを用いる方法」と「コーパスに基づく統計的な方法」の2種類についてまとめる。
*   それぞれの方法の**長所・短所をまとめる**。
*   それぞれの方法の**具体例を挙げながら説明する**。

#### **【課題2】**
*   湖における魚A～Fの出現確率 P(A)～P(F) を推定し、結果を比較する。
*   以下の捕獲結果を使用する：
    *   1回目： A 3匹、 B 6匹、 C 9匹、 D 0匹、 E 2匹、 F 0匹
    *   2回目： A 2匹、 B 15匹、 C 12匹、 D 0匹、 E 0匹、 F 1匹
    *   合計： A 5匹、 B 21匹、 C 21匹、 D 0匹、 E 2匹、 F 1匹
*   以下の方法で出現確率を推定し、結果を比較する：
    1.  **最尤推定**
    2.  **加算法（δ＝1）**
    3.  **削除推定法**
*   **削除推定法における補正出現回数には、以下の式を用いる**こと：
    *   𝐶𝑟01 / 𝑁𝑟0 + 𝐶𝑟10 / 𝑁𝑟1
    *   （講義第8回の資料にあった C_r01 + C_r10 / N_r0 + N_r1 の式は使用しない。この式ではN_r = 0 となる頻度 r があるときに確率の合計が1とならないため。）
