# プログラミング３用リポジトリ

## メモ


- ### PNG形式からEPS形式に変換

convert task.png task.eps

- ### tex のコンパイルの仕方

  platex ファイル名.tex  
  dvipdfmx ファイル名.dvi

  参考資料  
  https://www.ritsumei.ac.jp/~mmr14135/johoWeb/unix05.html

- ### tar.gz 圧縮の方法
  
  「XXXX.tar.gz」として、tar.gz圧縮アーカイブ化する場合
  tar -zcvf XXXX.tar.gz [File1] [File2] [File3]
