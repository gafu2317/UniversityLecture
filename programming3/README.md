# プログラミング３用リポジトリ

## メモ


- ### PNG形式からEPS形式に変換

convert task.png task.eps

- ### tex のコンパイルの仕方

  platex ファイル名.tex  
  dvipdfmx ファイル名.dvi

  参考資料  
  https://www.ritsumei.ac.jp/~mmr14135/johoWeb/unix05.html

- ### tar.gz 圧縮の方法
  

  tar -zcvf 35714121.tar.gz 35714121 
  (必要なフォルダを先に35714121に入れておく)
